# IcoChan

IcoChanは、複数の解像度を1つのファイルにまとめたWindows用マルチフレームアイコン（.ico）を簡単に作成できるデスクトップアプリケーションです。

---

## 🚀 主な機能

- **マルチフレーム対応**: 16x16から256x256までの標準的な解像度を網羅。
- **ドラッグ＆ドロップ操作**: ベース画像をドロップするだけで全サイズを自動生成。
- **個別カスタマイズ**: 特定の解像度だけに別の画像を割り当てることが可能。
- **透過確認モード**: ダークモード/ライトモードを切り替えて、アイコンの透過具合をチェック。
- **モダンなUI**: 直感的で使いやすいインターフェース。

## 📚 使い方

### 📖基本的な使い方

1. **ベース画像の読み込み**: アプリ中央のドロップゾーンに画像をドラッグ＆ドロップします。
2. **解像度の選択**: 出力したい解像度のチェックボックスをオンにします。
3. **（任意）個別画像の差し替え**: 特定のサイズのアイコン部分に別の画像をドロップすると、そのサイズだけを差し替えられます。
4. **透過の確認**: 右上の「Dark Background」スイッチで背景色を切り替え、透過が正しく処理されているか確認します。
5. **書き出し**: 「Export Multi-Frame Icon」ボタンを押し、保存先を指定して書き出します。

## 補足事項

### 🎨 注意事項
このアプリケーションは **PNG-encoded ICO** のみをサポートしています。  
古いICOフォーマット（RLE圧縮など）はサポートしていません。  
作られるアイコンはWindows Vistaより古いバージョンのWindowsでは認識しません。
